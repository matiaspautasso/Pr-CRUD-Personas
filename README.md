# Pr-CRUD-Personas

El proyecto es una API REST en Node.js que permite gestionar invitados de eventos mediante operaciones CRUD, persistiendo los datos en un archivo JSON.

El stack tecnológico incluye Node.js, módulos ES, y el sistema de archivos local.

La lógica de negocio permite alta, consulta, modificación y baja de invitados, asegurando nombres y apellidos latinos únicos y exponiendo endpoints accesibles desde cualquier frontend.
